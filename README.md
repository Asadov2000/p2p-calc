# P2P Calculator

Современный калькулятор для расчёта прибыли при P2P операциях с криптовалютой.

## Возможности

- Адаптивный дизайн для всех устройств
- Расчёт курса и прибыли
- История операций
- Статистика
- Тёмный режим
- Мультиязычность (RU/EN/TJ)
- Экспорт в PDF
- Учёт комиссий
- PWA и офлайн режим
- Интеграция с Telegram WebApp

## Установка

```bash
git clone https://github.com/Asadov2000/p2p-calc.git
cd p2p-calc
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Использование

1. Введите сумму в рублях
2. Укажите количество USDT
3. Система посчитает курс
4. Введите цену продажи для расчёта прибыли

## Технологии

- React 18 + TypeScript
- Tailwind CSS
- Zustand
- Vite
- Telegram WebApp SDK
